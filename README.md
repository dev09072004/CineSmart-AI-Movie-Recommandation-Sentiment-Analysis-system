# 🎬 CineSmart – AI Movie Recommendation & Sentiment Analysis System

CineSmart is a smart web-based application that recommends movies based on user preferences and analyzes user feedback using Artificial Intelligence (AI) and Machine Learning (ML).

---
 Features

- 🎥 Movie recommendations based on **genre & language**
- 🌐 Real-time data using **TMDB API**
- 💬 User feedback collection system
- 🧠 Sentiment analysis using **AI (TextBlob)**
- 🤖 Sentiment prediction using **ML (Naive Bayes)**
- 📊 Feedback stored in **Excel file**
- ⚡ Fast and responsive UI

 Technologies Used

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Flask (Python)  
- **API:** TMDB (The Movie Database)  
- **AI:** TextBlob (Natural Language Processing)  
- **ML:** Scikit-learn (Naive Bayes)  
- **Database:** Excel (.xlsx)

 How It Works

1. User selects **genre and language**
2. System fetches movies using TMDB API
3. Movies are displayed with posters & ratings
4. User provides feedback
5. AI + ML analyze sentiment
6. Final sentiment is stored in Excel file

 Project Workflow

User Input → Movie Recommendation → Feedback → Sentiment Analysis → Data Storage
